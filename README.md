# AgentPay Global Hackathon

**Vibe-code the future of autonomous AI payments.**

A global hackathon by BSVA focused on agentic payments — AI agents autonomously making micro-payments on BSV to interoperate.

## Structure

```
APH/
├── web/                  # Next.js hackathon website
│   ├── app/              # Pages + API routes
│   ├── components/       # React components
│   └── public/           # Static assets
├── marketing/            # Marketing materials
│   ├── content-calendar.md
│   └── copy/             # Pre-written social + email copy
├── templates/            # Starter templates for participants
│   ├── minimal/
│   ├── agent/
│   └── full-stack/
└── README.md
```

## Website Development

```bash
cd web
npm install
npm run dev    # http://localhost:3000
npm run build  # Production build
```

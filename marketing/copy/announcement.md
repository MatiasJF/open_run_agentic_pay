## LinkedIn Announcement

We're excited to announce the AgentPay Global Hackathon — a global vibe-coding competition challenging developers to build the first generation of autonomous AI agent payment systems.

The premise: use whatever AI coding tools you love to build an application where two or more AI agents discover each other, negotiate, and exchange value through BSV blockchain micro-payments — solving a real-world problem.

This hackathon is fully agent-agnostic. Use Claude, ChatGPT, Gemini, Copilot, Cursor, Windsurf, open-source models, or any combination. We don't care what tools you use — we care what you build.

Why this matters: As AI agents become more capable, they need native payment rails to transact autonomously. BSV's micro-transaction capability (sub-cent fees, instant settlement) is the ideal infrastructure for machine-to-machine economies. And with today's AI coding tools, a solo developer can build what used to take a team.

Key details:
- Registration: February 24 – March 13, 2026
- Hacking period: March 15–22 (8 days)
- Results ceremony: March 28
- Open to individuals and teams (1–4 members)
- Free to participate
- Any AI coding tool welcome

We provide the @bsv/simple library, an MCP server, starter templates, and complete documentation. The rest is up to you and your AI.

## Dev.to / Reddit / HN Post

**Title:** AgentPay: A vibe-coding hackathon where AI agents pay each other

Hey everyone,

BSV Blockchain Association is running a global vibe-coding hackathon with a unique angle: use your favorite AI coding tools to build applications where AI agents autonomously make micro-payments to each other.

**The challenge:** Create an app where 2+ AI agents discover each other, negotiate, and exchange value through BSV blockchain micro-payments. The agents need to transact autonomously — no human clicking "send payment."

**Fully agent-agnostic:** Use whatever AI tools you want for building — Claude, ChatGPT, Cursor, Copilot, Windsurf, Gemini, local LLMs, whatever. For the agents themselves, same deal — any model, any framework. We only care that they pay each other on BSV.

**Why BSV?** Sub-cent transaction fees make it viable for agents to pay each other fractions of a penny per query, per task, or per data point. Thousands of micro-payments for less than a cent in total fees.

**What's provided:**
- `@bsv/simple` — modular JS/TS library for wallets, payments, tokens, credentials, messaging
- MCP server — gives any compatible AI assistant deep BSV knowledge
- Starter templates in Next.js
- BSV Desktop Wallet for funding

**Format:** Solo vibe-coders or teams (1-4). Free entry. Global.

**Timeline:**
- Registration: Feb 24 – Mar 13
- Hacking: Mar 15 – 22 (8 days)
- Results: Mar 28

**Project ideas:** Data marketplace agents, AI service brokers, content licensing networks, distributed compute settlement, IoT data economies — or anything you can think of.
